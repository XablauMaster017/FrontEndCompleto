Exemplo — Um Programa Simples

Aqui está um pequeno programa que pede dois números ao usuário e exibe a soma deles.

soma.cpp
#include <iostream>
using namespace std;

// Função para somar dois números
int somar(int a, int b) {
    return a + b;
}

int main() {
    int x, y;

    cout << "Digite dois números: ";
    cin >> x >> y;

    int resultado = somar(x, y);

    cout << "Soma = " << resultado << endl;

    return 0;
}
1

#include <iostream> — carrega a biblioteca necessária para entrada e saída de dados (cin e cout).

2

int somar(int a, int b) — uma função que recebe dois inteiros e retorna a soma deles.

3

int main() — o ponto de entrada de todo programa C++. A execução começa aqui.

4

cin >> x >> y — lê dois valores digitados pelo usuário no teclado.

5

return 0 — informa ao sistema operacional que o programa terminou sem erros.

O Começo

Siga estes passos em ordem e em breve você estará escrevendo programas de verdade.

1. Instale um Compilador

Baixe o GCC (MinGW no Windows) ou use um compilador online como o onlinegdb.com para rodar código sem instalar nada.

2. Escreva o "Hello World"

Comece com o programa mais simples possível. Certifique-se de compilar e executar antes de avançar.

3. Aprenda Variáveis e Laços

Pratique guardar valores e repetir ações. Esses dois conceitos são a base de qualquer programa.

4. Escreva Funções

Divida seu código em partes pequenas e reutilizáveis. Isso torna os programas mais fáceis de ler e corrigir.

5. Crie um Pequeno Projeto

Una tudo em um programa — uma calculadora, um jogo de adivinhar número ou uma calculadora de média escolar.
